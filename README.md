# 🍔 Cardápio Digital - Dev Burguer

Este é um projeto de **Cardápio Online** responsivo, desenvolvido para facilitar a visualização de produtos e a realização de pedidos via WhatsApp[span_1](start_span)[span_1](end_span). A aplicação permite que o utilizador navegue pelas opções, adicione itens ao carrinho e envie o pedido diretamente para o estabelecimento[span_2](start_span)[span_2](end_span).

## 🚀 Funcionalidades

*   **🖼️ Visualização de Produtos:** Lista de itens com imagem, descrição e preço[span_3](start_span)[span_3](end_span).
*   **🛒 Sistema de Carrinho:** Adicionar e remover produtos, com atualização dinâmica do total[span_4](start_span)[span_4](end_span).
*   **🕒 Horário de Funcionamento:** Verificação em tempo real se o estabelecimento está aberto ou fechado[span_5](start_span)[span_5](end_span).
*   **📱 Integração com WhatsApp:** Envio automático dos detalhes do pedido (itens, total e morada) via API do WhatsApp[span_6](start_span)[span_6](end_span).
*   **📍 Validação de Morada:** Alerta o utilizador caso tente finalizar o pedido sem informar o local de entrega[span_7](start_span)[span_7](end_span).
*   **📱 Design Responsivo:** Adaptado para dispositivos móveis e desktop[span_8](start_span)[span_8](end_span).

## 🛠️ Tecnologias Utilizadas

*   **🌐 HTML5:** Estrutura da página[span_9](start_span)[span_9](end_span).
*   **🎨 Tailwind CSS:** Estilização moderna e responsiva[span_10](start_span)[span_10](end_span).
*   **⚡ JavaScript (ES6+):** Lógica do carrinho, manipulação do DOM e integração com API[span_11](start_span)[span_11](end_span).
*   **🔔 Toastify JS:** Notificações flutuantes para melhorar a experiência do utilizador[span_12](start_span)[span_12](end_span).
