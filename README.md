# 🍔 Cardápio Digital - Dev Burguer

Este é um projeto de **Cardápio Online** responsivo, desenvolvido para facilitar a visualização de produtos e a realização de pedidos via WhatsApp. A aplicação permite que o utilizador navegue pelas opções, adicione itens ao carrinho e envie o pedido diretamente para o estabelecimento.

## 🚀 Funcionalidades

- **Visualização de Produtos:** Lista de itens com imagem, descrição e preço.
- **Sistema de Carrinho:** Adicionar e remover produtos, com atualização dinâmica do valor total.
- **Horário de Funcionamento:** Verificação em tempo real (badge dinâmico) se o estabelecimento está aberto ou fechado.
- **Integração com WhatsApp:** Envio automático dos detalhes do pedido (itens, total e morada) via API do WhatsApp.
- **Validação de Morada:** Alerta o utilizador caso tente finalizar o pedido sem informar o local de entrega.
- **Design Responsivo:** Interface totalmente adaptada para dispositivos móveis e desktop.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica da página.
- **Tailwind CSS:** Estilização moderna, utilitária e rápida.
- **JavaScript (ES6+):** Lógica do carrinho, manipulação do DOM e integração com API.
- **Toastify JS:** Notificações flutuantes elegantes para feedback do utilizador.
